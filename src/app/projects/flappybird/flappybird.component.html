Flappy Bird AI
Goal: Build a game from scratch that is able to play itself
Technologies used: Java, NEAT
When: Feb 2020
Helpful Links: GitHub

Overview
This is a clone of flappy bird and my first try at game development from scratch. I originally created this flappy bird for a communications class in my second semester are college. The class project was about unfamiliar genres, hence my first try atmaking a game. The game is written in Java and the engine is one that I created as well. After the class, Iwas still interested in taking it a bit further, so I hooked a machine-learning algorithm to it so it couldplay for me. The ML algorithm I went with was NEAT as it seemed like a reasonable place to start with machinelearning and AI.
